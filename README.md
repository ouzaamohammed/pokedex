A REPL program that simulate what Pokedex do in Pokemon world.

## Usage

First, clone the repo and cd into it

```bash
git clone https://github.com/ouzaamohammed/pokedex
cd pokedex
```
then make sure you're using the right version of node using `nvm`

```bash
nvim use
```

if you don't have nvm, install it from [here](https://github.com/nvm-sh/nvm?tab=readme-ov-file#installing-and-updating).

install dependencies and run the program

```bash
npm i
npm run dev
```

you get a prompt run the help command

```
pokedex > help
```
